Sprawdź czy plik jest ignorowany:
   git check-ignore -v nazwa_pliku

Sprawdź wszystkie ignorowane pliki:
   git status --ignored

Sprawdź co będzie commitowane:
   git status

Sprawdź czy nie ma sekretów w zmianach:
   git diff | grep -i "password\|secret\|key\|token"

╔══════════════════════════════════════════════════════════════════════════════╗
║                    🚨 JEŚLI PRZYPADKOWO DODAŁEŚ SEKRET                       ║
╚══════════════════════════════════════════════════════════════════════════════╝

NIE commitowałeś:
   git reset HEAD plik_z_sekretem
   git checkout -- plik_z_sekretem

Commitowałeś (NIE pushowałeś):
   git reset --soft HEAD~1

Pushowałeś:
   1. ZMIEŃ SEKRET NATYCHMIAST!
   2. Użyj git-filter-repo lub BFG Repo-Cleaner
   3. Force push (git push --force)
