🚀 Quick Start

1. Konfiguracja

Upewnij się, że plik .env jest poprawnie skonfigurowany z tokenem Telegram i API Key.

2. API Key

Wszystkie requesty do API wymagają headera: X-API-Key: twój_api_key

3. Sprawdź Health

Odwiedź http://localhost:8000/health aby sprawdzić status systemu
