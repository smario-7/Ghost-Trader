🔧 OPCJA 1: Automatyczna aktywacja przez direnv (ZALECANE)
   
   1. Zainstaluj direnv:
      Ubuntu/Debian: sudo apt install direnv
      macOS: brew install direnv
   
   2. Dodaj do ~/.bashrc:
      echo 'eval "$(direnv hook bash)"' >> ~/.bashrc
      source ~/.bashrc
   
   3. W katalogu projektu:
      direnv allow
   
   ✨ Od teraz środowisko aktywuje się automatycznie!

🔧 OPCJA 2: Ręczna aktywacja przez skrypt
   
   source activate.sh
   
   Skrypt automatycznie:
   - Sprawdzi czy conda jest zainstalowana
   - Utworzy środowisko jeśli nie istnieje
   - Aktywuje środowisko

🔧 OPCJA 3: Standardowa aktywacja conda
   
   conda activate ghost-trader

🔧 OPCJA 4: Przez Makefile
   
   make activate  # Pokaże instrukcje
   make setup     # Utworzy środowisko
   make install   # Zainstaluje zależności

Sprawdź czy wszystko działa:

1. Aktywuj środowisko:
   source activate.sh

2. Sprawdź wersję Python:
   python --version
   → Powinno pokazać: Python 3.11.13

3. Sprawdź ścieżkę:
   which python
   → Powinno pokazać: /home/mario/miniconda3/envs/ghost-trader/bin/python

4. Sprawdź pakiety:
   pip list
   → Powinny być wszystkie z requirements.txt
